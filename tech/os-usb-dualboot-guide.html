Guide To Dual-Booting, USB Boots, And Changing Your OS

By Firethorn

I am not responsible for damages done to any device, whether in hardware or software.

  1. Preparation

    Make backups of any important files before attemping anything in this guide. In addition, you will need at least one USB (empty or one you're okay with deleting everything from, at least 8 GB too) and an ISO file for the OS of your choice.

    You may shuffle files around as needed to gain space. ISO downloads for a few Linux OSs will be included at the bottom of the page.

  3. The Boot Menu

    If this page is loaded on the target computer, transfer it to another device before proceeding.

    Completely turn off your computer and plug in the USB.

    When rebooting, look for a message (often in the bottom left of the screen) telling you to press a key for a menu. If there's an option to go directly to the boot menu, press that key. Otherwise, press the key for the the BIOS menu and navigate to the boot menu from there.

    With some inferences and guesswork, figure out what option boots to your USB. It's okay if you have to reboot a few times to get it.

    *If you're on a Chromebook, this is very different. Follow this tutorial instead.

    *On a Mac, you have to hold either Option or Power, depending on your device.

  4. Installation

    If you intend to boot only from a USB, then congratulations! Whatever you did on your last boot is what you need to do to run your OS of choice. You may unplug the USB after shutting down and, if you wish, change the boot order in your BIOS to use a USB boot if available. Otherwise, continue reading.

    Most ISOs either prompt you to install immediately or have an installation program on the desktop. If you wish to dual-boot, make sure to tell it to only use part of your hard drive (Your system will ask which OS to use on startup). If this option is not available, return to your main OS and partition your hard drive. Backups are crucial here.

    If you do not wish to dual-boot and instead want to completely change your OS, simply tell the program to do that instead. If you choose this, you entire hard drive will be overwritten.

Linux OS List
  • Linux Mint - Popular, simple, and functional. Similar to Windows.
  • MX Linux - Similar to Windows.
  • Ubuntu - A very popular Linux OS with custom themes, long-term support, and a huge community.
  • Xubuntu - Like Ubuntu, but the initial setup is lighter on disk space.
  • Kali Linux - Commonly used by hackers and pen-testers.
  • Tails OS - Designed to be booted from a USB so you can run Tor without leaving a trace on the hard drive.
  • Arch Linux - In-depth and DIY, has a user-owned repository. Very popular.
  • Debian Linux - The oldest Linux distribution. Stable and dependable with great suppport.
  • Fedora - Very popular.
  • Zorin OS - Similar to Windows and MacOS. Easy to use.
  • Solus - Plug-and-play, great for developers too.
