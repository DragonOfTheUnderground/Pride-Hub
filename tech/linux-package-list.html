Linux Package List

By Firethorn
  • Firefox - sudo snap install firefox - Non-chromium browser, my favorite for desktop.
  • Geany - sudo apt install geany - Code editor and compiler, works well even on a Pi.
  • Raspberry Pi Imager - sudo apt install rpi-imager - Tool for loading an os for a Pi onto a removable drive.
  • Thonny - sudo apt install thonny - Python shell and editor.
  • Games - sudo apt install (bastet, freesweep, nsnake, greed, xbomb, nudoku) - Various games.
  • BSD Games - sudo apt install bsdgames - More games.
  • Freecad - sudo apt install freecad - CAD program.
  • Pinta - sudo apt install pinta - Image editor, alternative to GIMP.
  • HTTrack - sudo apt install webhttrack - Website copier. Be careful not to accidentally download too much.
  • VLC - sudo apt install vlc - Multimedia player, available for pretty much any major OS.
  • Webcamoid - sudo apt install webcamoid - Camera app.
  • Audacity - sudo apt install audacity - Audio recorder and editor.
  • Amfora - CLI browser for the gemini:// protocol.
  • Lagrange - GUI browser for the gemini:// protocol.
  • Tor Browser - Browser that proxies traffic to avoid censorship. For the dark web, it's best to use Tor through Tails OS instead.
  • Balena Etcher - Tool to put ISOs onto USBs.
  • GParted - sudo apt install gparted - Tool for managing disk and drive partitions.
  • PrusaSlicer - Slicer for 3D printing.
  • Auto-Py-To-Exe - sudo pip install auto-py-to-exe - Tool to convert .py files to .bin executables. The .bin extension is not added automatically.
